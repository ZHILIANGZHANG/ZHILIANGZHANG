# 个人学术主页美化设计文档

**日期**: 2026-04-06  
**主题**: 学术个人主页视觉升级  
**风格定位**: 温润质感 · 学术简约

---

## 1. 设计目标

在保持学术简约风格的基础上，通过精致的排版、考究的色彩和合理的留白，提升页面的高级感与专业度。

**核心原则**:
- 内容优先，设计服务于信息传达
- 克制使用视觉元素，避免过度设计
- 保持学术界的审美惯例

---

## 2. 视觉系统

### 2.1 色彩方案

| 用途 | 颜色值 | 说明 |
|------|--------|------|
| 背景主色 | `#FAFAF8` | 极浅米白，温润基调，替代纯白减少刺眼感 |
| 背景次色 | `#FFFFFF` | 纯白，用于内容区域对比 |
| 文字主色 | `#1A1A1A` | 近黑色，保证阅读舒适度 |
| 文字次色 | `#666666` | 中灰，用于辅助信息、日期等 |
| 文字弱化 | `#999999` | 浅灰，用于 Section 标题 |
| 强调色 | `#2563EB` | 学术蓝，所有高亮、链接、标签 |
| 边框/分隔线 | `rgba(0,0,0,0.08)` | 极浅灰，用于分隔线 |

### 2.2 字体系统

**字体家族**: `Inter, -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif`

| 元素 | 字号 | 字重 | 字间距 | 行高 | 颜色 |
|------|------|------|--------|------|------|
| 姓名 (H1) | 48px | 700 | -0.03em | 1.2 | #1A1A1A |
| 副标题 | 18px | 400 | -0.01em | 1.5 | #666 |
| 所属机构 | 15px | 400 | 0 | 1.6 | #444 |
| Section 标题 | 13px | 600 | 0.08em | 1.4 | #999 |
| 正文 | 15px | 400 | 0 | 1.8 | #444 |
| 论文标题 | 16px | 600 | -0.01em | 1.4 | #1A1A1A |
| 作者/日期 | 14px | 400 | 0 | 1.5 | #666 |
| 标签 | 13px | 500 | 0.01em | 1.4 | #2563EB |
| 联系链接 | 14px | 500 | 0 | 1.5 | #666 |

### 2.3 间距系统

**页面布局**:
- 最大宽度: `1000px`（从原 880px 增加）
- 页面内边距: `24px`（移动端）
- Section 间距: `72px`

**内容间距**:
- 条目内部: `20-24px`
- 紧凑信息组: `12-16px`
- 标签间隙: `8px`

## 3. 组件样式

### 3.1 导航栏

- 固定定位，顶部
- 背景: `rgba(250, 250, 248, 0.95)` + `backdrop-filter: blur(20px)`
- 底部边框: `1px solid rgba(0,0,0,0.06)`
- 高度: `64px`
- 品牌名: 17px, 字重 600, 字间距 -0.02em

### 3.2 Hero 区域

- 布局: Flex 横向排列，头像 + 内容
- 头像: 160px × 160px，圆形，外围 2px 浅灰细环
- 内容区: 与头像间距 48px
- 姓名与副标题间距: 8px
- 副标题与机构间距: 16px
- 机构与联系链接间距: 24px

### 3.3 Section 标题

- 文字: 大写，字间距 0.08em
- 底部边框: 1px solid rgba(0,0,0,0.08)
- 内边距: 底部 12px
- 与内容间距: 24px

### 3.4 研究兴趣标签

- 背景: `rgba(37, 99, 235, 0.08)`（学术蓝 8% 透明度）
- 文字: `#1D4ED8`（学术蓝深色）
- 圆角: `16px`
- 内边距: `6px 14px`
- 字号: `13px`
- 字重: `500`

### 3.5 新闻条目

- 布局: Flex，年份左对齐（56px 宽）+ 内容区
- 年份: 14px, 字重 600, 颜色 #999
- 内容: 15px, 颜色 #333
- 日期: 13px, 颜色 #999, 左侧间距 8px
- 条目间距: 16px 上下
- 分隔线: 1px solid rgba(0,0,0,0.05)

### 3.6 论文条目

- 标题: 16px, 字重 600, 颜色 #1A1A1A
- 作者: 14px, 颜色 #666，本人名字加粗
- 会议: 14px, 斜体, 颜色 #888
- 链接: 13px, 颜色 #666, hover 变学术蓝
- 条目间距: 24px 上下
- 分隔线: 1px solid rgba(0,0,0,0.06)

### 3.7 教育/经历条目

- 布局: Flex，Logo（48px）+ 内容区
- 标题与日期: Flex 两端对齐
- 标题: 15px, 字重 600
- 日期: 13px, 颜色 #999
- 地点: 14px, 颜色 #666
- 详情: 14px, 颜色 #888

## 4. 交互设计

页面整体静态，仅保留基础 hover 状态：

| 元素 | 默认状态 | Hover 状态 | 过渡时长 |
|------|----------|------------|----------|
| 导航链接 | #666 | #1A1A1A | 200ms |
| 内容链接 | #1A1A1A | 学术蓝 + 下划线 | 200ms |
| 论文标题 | #1A1A1A | #2563EB | 200ms |
| 联系图标 | #666 | #2563EB + 上浮 2px | 200ms |

**无页面加载动画、无滚动触发动画。**

## 5. 响应式设计

### 断点: 768px

**移动端适配**:
- Hero 区域改为垂直排列，居中对齐
- 导航链接隐藏，显示汉堡菜单按钮
- 新闻条目改为垂直堆叠（年份在上）
- 经历条目 Logo 与内容垂直排列
- 联系链接换行居中
